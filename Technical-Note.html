Technical Note: Context-based grading



1 Introduction


This technical note documents a Shiny-based application developed to assess students grades in a context-aware fashion. It integrates socioeconomic, educational, and logistical factors to rescale and rank academic performance, aiding pedagogical and sociological interpretation.


2 - Rescaling Function


The core transformation relies on linear algebra and multivariate statistics. The function rescales a target variable (e.g., grade) using the Mahalanobis-like transformation.

Given a dataset \(X \in \mathbb{R}^{n \times 2}\), where one column is the contextual variable and the other is the score:
1. Compute the covariance matrix \(\Sigma\) of \(X\)
2. Cholesky Decomposition: \(\Sigma = L L^T\)
3. Compute \(L^{-1}\) (pseudo-inverse)
4. Transform: \(X_{rescaled} = X L^{-1}\)

Then rescale the transformed score to original bounds:

\[ \text{nv}_{rescaled} = \left( \frac{nv - \min(nv)}{\max(nv) - \min(nv)} \right) (\max(x) - \min(x)) + \min(x) \]

An alternative version,forces rescaling between 0 and 10.

It is important to note that the above formulation may be modified in future editions.
